# External Material(fusion, keyshot, ect) Three Visual Regression Testing Harness

**To be used In Conjunction with KMP-Three-Suite**

A tool For measuring and refining the parameters and effects of material, shader, and texture properties and how they are applied in the Threejs Canvas. Specifically when trying to translate materials and textures from external rendering applications, to visually match.

## MUST Haves 

    - Be able to recreate, render, and apply custom materials, capture them in screen shots and notice and define differences.

    - Compare Photos and recognize regression over time and in realtime. 

    - Must be able to Detect the properties in the material and how they are applied in the canvas.

    - Must Be able to compare recreated screenshots of objects in external renderers(standardized and provided ahead of the testing), and 
    apply the position of object using diff comparisons and then run the tests to compare material application and tune.
